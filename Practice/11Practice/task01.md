### Task: Install dbt and build a mart model with aggregations

_1 point_

Install and run the jaffle_shop_duckdb project, then create your own mart-layer dbt model that contains at least one aggregation. The repository is a self-contained dbt playground based on fictional ecommerce data, and it includes seeds plus starter models for customers, orders, and payments. **_It requires Python 3.13 or higher._**

#### How to install dbt
Follow instructions here https://github.com/dbt-labs/jaffle_shop_duckdb. 
You also can use **_my_commands_to_install_dbt.md_** for Mac OS.
